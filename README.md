# Tauá 🪨

An experimental Wayland Compositor

## Why

My plastic Wayland compositor, tauá means clay in Tupi, and this is a project where I can do whatever I want on screen, not limited by most window managers conventions.

## Development

All code in this project was written by hand as a learning exercise. Claude was used as an interactive tutor to explain Wayland and smithay concepts, answer questions, and point out errors -- not to generate code.
